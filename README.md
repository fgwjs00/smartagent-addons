# SmartAgent Add-ons

Current release: `Beta 0.0.6`

## 中文说明

本仓库提供 SmartAgent 的 Home Assistant add-on 发布包。用户在 Home Assistant Add-on Store 中添加本仓库后，可以安装和升级 `SmartAgent AI Core`。

SmartAgent AI Core 承载 SmartAgent Gateway、管理控制台、中控屏资源、本地 AI 服务和 Local Core 相关能力入口，用于把 SmartAgent 的产品能力运行在 HA OS / Home Assistant 项目环境中。

## English

This repository provides the Home Assistant add-on package for SmartAgent. After adding this repository to the Home Assistant Add-on Store, users can install and upgrade `SmartAgent AI Core`.

SmartAgent AI Core serves the SmartAgent Gateway, management console, central control screen assets, local AI service, and Local Core related entry points so SmartAgent can run inside an HA OS / Home Assistant project environment.
